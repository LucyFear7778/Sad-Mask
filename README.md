# SAD-MASK v2.5
A URL masking tool for ethical hacking and security awareness  🌟

## Features
- 🛡️ Mask URLs using fake domains
- 📱 Generate QR codes
- 💾 Save masked links to JSON file
- 🎨 Colorful PH flag-themed banner
